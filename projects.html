I designed my personal website using html + css. I kept the overall theme modern and minimalistic. Read all about my projects, community work and favourtie hobbies! During the NASA Space-Apps hackathon, my team created a website using HTML, CSS and Javascript that used Nasa’s Big Data and weather API’s to help control the spread of airborn diseases and inform cities of their out-break risk. During the virtual ‘RU HACKS’, my team tackled the category of harm prevention. Using Fusion360, java and FramerX, we created a software that detected blood-alcohol levels in sweat using a wristband and alerted the user through a mobile device when they were reaching thier substance limits. As a entry into RBC’s ‘Next Great Innovator’ hackathon, my team and I created a chrome extension using html, css, and figma that reccomended more sustainable clothing items before the user made a purchase. ‘Healthy-Grow’ is a protective cover that shields the roots of plants from hazardous threats such as pollution, microplastics and potential harm from animals. The product was made entirely from shoelaces, glue and papeclips and was an entry in Deca Idea Challenge 2019.
